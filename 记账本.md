# **💰 随手记账 (Expense Tracker)**

一个轻量级、无需安装、基于浏览器本地存储的个人记账应用。

🔗 **在线演示**: \[点击这里查看你的网页链接\] (部署后将你的 GitHub Pages 链接填在这里)

## **✨ 功能特点**

* **零安装**：单文件 HTML，双击即用，也可部署在任何静态网页托管服务上。  
* **数据安全**：所有数据仅存储在您的浏览器本地 (LocalStorage)，不会上传到任何服务器，隐私绝对安全。  
* **收支管理**：轻松记录每一笔收入和支出。  
* **图表分析**：自动生成甜甜圈图，直观展示各类消费占比。  
* **响应式设计**：完美适配手机和电脑端，手机端拥有原生 App 般的底部导航栏。  
* **智能分类**：内置餐饮、交通、购物等多种常用分类，并以不同颜色区分。

## **🛠️ 技术栈**

* **React 18**: 用于构建用户界面。  
* **Tailwind CSS**: 用于现代化的样式设计。  
* **Babel (Standalone)**: 支持在浏览器中直接运行 JSX 代码。  
* **LocalStorage**: 用于数据持久化。

## **🚀 如何使用**

### **方法一：直接运行 (推荐)**

1. 下载本项目中的 index.html 文件。  
2. 直接双击打开即可使用。

### **方法二：部署到 GitHub Pages**

1. Fork 本仓库或下载文件上传到你自己的 GitHub 仓库。  
2. 在仓库 Settings \-\> Pages 中开启 GitHub Pages。  
3. 访问生成的链接即可在手机或电脑上随时记账。

## **📝 注意事项**

* **数据同步**：由于使用本地存储，更换浏览器或设备后，之前的记录**不会**同步。请尽量固定使用同一设备的一个浏览器进行记账。  
* **清理缓存**：请勿随意清理浏览器的“网站数据/缓存”，否则可能导致记账记录丢失。
